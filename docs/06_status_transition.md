# 状態遷移表

## 1. ステータス定義

| ステータス | 表示名 | 概要 |
|---|---|---|
| `DRAFT` | 下書き | 申請前の状態。申請者が編集・削除できる。 |
| `SUBMITTED` | 申請中 | 承認待ちの状態。申請者は編集・削除できない。 |
| `APPROVED` | 承認済み | 承認が完了した状態。 |
| `RETURNED` | 差戻し | 承認者から差戻された状態。申請者が編集・再申請できる。 |

## 2. 状態遷移

| 現在ステータス | 操作 | 次ステータス | 実行可能者 | 条件 |
|---|---|---|---|---|
| なし | 作成 | `DRAFT` | 認証済みユーザー | 明細が 1 件以上存在する。 |
| `DRAFT` | 更新 | `DRAFT` | 申請者本人 | 他ユーザーは不可。 |
| `RETURNED` | 更新 | `DRAFT` | 申請者本人 | 差戻し情報をクリアする。 |
| `DRAFT` | 削除 | 削除 | 申請者本人 | 他ユーザーは不可。 |
| `RETURNED` | 削除 | 削除 | 申請者本人 | 他ユーザーは不可。 |
| `DRAFT` | 申請 | `SUBMITTED` | 申請者本人 | 申請日時を設定する。 |
| `RETURNED` | 申請 | `SUBMITTED` | 申請者本人 | 差戻し情報をクリアする。 |
| `SUBMITTED` | 承認 | `APPROVED` | APPROVER / ADMIN | 自分の申請は不可。 |
| `SUBMITTED` | 差戻し | `RETURNED` | APPROVER / ADMIN | 自分の申請は不可。差戻し理由必須。 |

## 3. 禁止遷移

| 現在ステータス | 操作 | 理由 |
|---|---|---|
| `SUBMITTED` | 更新・削除 | 承認待ちのため申請者による変更不可。 |
| `APPROVED` | 更新・削除・申請 | 承認済みのため変更不可。 |
| `APPROVED` | 承認・差戻し | 承認待ちではないため不可。 |
| `DRAFT` | 承認・差戻し | 申請中ではないため不可。 |
| `RETURNED` | 承認・差戻し | 申請中ではないため不可。 |

## 4. 状態遷移図

```text
            作成
             ↓
          DRAFT
       更新 ↓ 申請
             ↓
        SUBMITTED
        ↓       ↓
      承認     差戻し
        ↓       ↓
     APPROVED  RETURNED
                 ↓
               更新
                 ↓
               DRAFT
```
